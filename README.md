# Introduction to Engineering Design
### The Cushman School — Willie Avendano

*PLTW IED*

Students apply the design process to model and prototype solutions using 3D modeling software. The course emphasizes iterative design, technical documentation, and the connection between physical and digital fabrication.

## Course Overview

- 3D solid modeling with Autodesk Inventor
- Geometric dimensioning and tolerancing (GD&T)
- Reverse engineering
- Technical drawing and documentation
- Prototype fabrication and evaluation

## Tools & Software
- Autodesk Inventor
- 3D printers (FDM)
- PLTW Activity Central
